## 상시 고정 수칙 (Standard Operating Procedure)

1. **[Side-Effects] 파급효과 교차 체크 의무**: 하나의 변경을 진행하면 반드시 "모바일/데스크톱 뷰포트 크로스 체크"와 "메인/목록/상세페이지 컴포넌트 세트 체크"를 동시 수행한다.
2. **한/영 매칭 유연화**: 한/영 글은 1:1 강제 매칭을 하지 않고 타겟 오디언스에 맞춰 독자적으로 로컬라이징(동기화 해제)하되, `translationKey`로 연동된 글만 정보 싱크를 맞춘다.
3. **이모지 금지**: 본문 내 이모티콘 사용은 절대 금지한다.
4. **빌드 검증**: 레이아웃이나 설정 수정 시 반드시 `hugo` 명령으로 로컬 빌드 성공 여부를 먼저 확인한다.
